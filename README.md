# マツダ危機一髪！ (Mazda Crisis!)

ビール樽に閉じ込められたマツダを救い出す、新感覚の連打アクションゲームです！

---

## 🎮 どんなギミックで動いているのか
本作はフロントエンド技術（HTML5 / CSS3 / JavaScript）のみで構築されています。
- **Canvas アニメーション**: HTML5 `<canvas>` を用いて、プレイヤーのポンプアクション、樽の揺れや泡パーティクル、マツダの表情変化、ヤギなどの滑らかなアニメーションをフレーム単位で制御しています。
- **動的UI制御**: CSSベースの体力・炭酸・耐久ゲージがゲームの状態に応じてリアルタイムに変動・変色し、緊急時には画面を赤く光らせるフラッシュエフェクト（再描画コストを抑えた設計）などを実装しています。
- **バックエンド通信連携**: ランキング機能はGoogle Apps Script (GAS) APIを通して非同期通信を行い、Googleスプレッドシート等へスコアを記録・取得します。

## 📖 ゲームストーリー
借金取りに追われ、絶体絶命のピンチに陥ったマツダ！
彼が身を隠した（あるいは閉じ込められてしまった）のは、大きなビール樽の中でした。
マツダを助けるための手段は一つ。樽の中の炭酸圧をポンプで高め、その勢いでフタごと借金取りを吹き飛ばすこと！
果たしてあなたは、自らの体力が尽きるより先にマツダを救出できるでしょうか？

## 👥 登場人物
- **マツダ**: ビール樽の中にいる主人公。プレイヤーの連打（炭酸）の溜まり具合や樽のヒビ割れ具合に応じて、様々な表情（焦り、絶望、歓喜など）を見せます。
- **プレイヤー**: 右下で懸命にポンプを操作する人物。無理な連打を続けるとヘトヘトになって倒れてしまいます。
- **借金取り**: マツダを追いつめた元凶。オープニングなどで登場します。
- **ヤギ（黒ヤギ・白ヤギ）**: 画面奥でなぜかじっと見守っている謎の存在。

## 📜 ゲームルール
ポンプボタン（「ポンプ！」）をクリック・連打して、炭酸ゲージをMAXにすればゲームクリア（救出成功）です。
ただし、以下の条件に気をつける必要があります。

1. **体力メーター**
   - ポンプを押すたびに体力が消費されます。
   - 休まず連打し続けて体力が `0` になると、「過労（力尽きた）」でゲームオーバーになります。
   - 体力はポンプを押していない間に自動で回復します。
2. **炭酸メーター**
   - ポンプを押すと増えますが、時間経過で自然に抜けて（減って）いきます。
   - ゲージが `100` に達した瞬間にクリアです。
3. **樽耐久メーター**
   - 短時間に猛烈なスピードで連打しすぎると、樽に負荷がかかり耐久度が減ります。
   - 耐久度が `0` になると「樽爆発」でゲームオーバーになります。

*💡休んで体力を回復させつつ、炭酸が抜けないように絶妙なペースで連打するのがコツです！*

## 💯 採点の仕方
ゲーム終了時（クリア/ゲームオーバー問わず）、あなたのプレイスタイルに応じたスコアが算出されます。
スコアは以下の要素の合計値です。

- **努力点**: 溜めた総炭酸量 × 5点
- **無駄な素振り点**: ポンプを押した総回数 × 20点
- **チキンレース点**: 体力や耐久度がギリギリのピンチ状態（赤ゲージ）で耐えた時間 × 2点
- **称号ボーナス**: プレイ結果に応じてユーモアあふれる称号とボーナス点が与えられます。
  - 【マツダの救世主】（クリア成功）: +10000点
  - 【破壊神RTA】（数秒で樽を破壊）: +1500点
  - 【秒速のもやし】（数秒で体力切れ）: +1500点
  - 【ギリギリを攻めすぎた愚者】（長時間のチキンレース）: +1000点
  - 【歩く大爆発】（樽破壊）: +500点
  - 【名誉ブラック社員】（体力切れ）: +500点

## 🐦 X（旧Twitter）への投稿の仕方
リザルト画面に表示される **「𝕏 で共有」** ボタンをクリックすると、Xの投稿画面が自動で開きます。
あなたの獲得した「スコア」と「称号」、そしてゲームを遊ぶためのURLがあらかじめテキストに入力された状態になるため、そのままポストするだけで簡単に結果をシェアできます。OGP画像によって、Xのタイムライン上でもマツダの姿が目立つように表示されます。

## 🏆 ランキング機能について
ゲーム内で獲得したスコアは、リーダーボード（ランキング）に登録して世界中のプレイヤーと競うことができます。
1. リザルト画面でユーザー名（10文字以内）を入力し、「スコアを登録」ボタンを押します。
2. データはGAS経由で自動的にデータベースに保存されます。
3. タイトル画面やリザルト画面の「ランキングを見る」ボタンを押すと、いつでも上位プレイヤーのスコアと名前、プレイ日時を確認できます。
